## Code Notes

- `src/main/agent/stream.ts` now returns the assistant message blocks generated by the stream so the loop can send valid `tool_result` follow-up messages.
- `src/main/agent/loop.ts` owns the terminal `done` status; `ipc.ts` only logs aggregate usage and maps thrown errors.
- `src/renderer/App.tsx` subscribes to `agent:event` and updates the existing Zustand agent slice.
- `src/renderer/components/CommandInput.tsx` uses the existing busy button as the stop button and maps Escape to `stopAgent` while a turn is active.
